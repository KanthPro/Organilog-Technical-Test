# Organilog – Technical Test

## Prérequis
- Node.js (version LTS recommandée)
- pnpm

## Installer pnpm (pour débutants)
Si pnpm n'est pas installé, vous pouvez l'ajouter globalement avec npm :
```bash
npm install -g pnpm
```
Vérifiez ensuite l'installation :
```bash
pnpm --version
```

## Installation
```bash
pnpm install
```

## Lancer en développement
```bash
pnpm dev
```

## Build production
```bash
pnpm build
```

## Prévisualiser le build
```bash
pnpm preview
```

## Lint
```bash
pnpm lint
```
